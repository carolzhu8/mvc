The most important part is the model class: first, it exposes all the important methods to outside.
second, it is an integrations of all inner functions.

Model is used to create, modify and store all kinds of shapes. plus, it can make shapes into snapshots.
It also provides methods for outside class to read snapshots.

In the shape package, there are Ishape interface that defines methods of all shapes. The
Abstractshape is a super class of all shapes.

In factory package, factory class is used to build all kinds of shapes instance.
